**Q: What is a Real Estate API?**
A: A Real Estate API (Application Programming Interface) allows different software systems to communicate. AgentClaw API specifically provides the "brain" for real estate automation, connecting your CRM to AI models, mailing services, and voice platforms.

**Q: How do I automate realtor follow-up calls?**
